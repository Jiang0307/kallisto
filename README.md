# kallisto

## Installation
* In kallisto/
```shell
chmod +x build.sh
sudo build.sh
```

## Usage : Build index table
* In kallisto/
```shell
chmod +x kallisto_index.sh
./kallisto_index.sh -k <k>
```

## Usage : Pseudoalignment
* In kallisto/
```shell
chmod +x kallisto_pseudo.sh
./kallisto_pseudo.sh -k <k>
```
